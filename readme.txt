How to use:

Register the dll then open a new VB project and add FlashBox as a reference. See the included Sample use project for source code on how to use it.

Make sure you keep the flashmsg.swf in the same directory as the registered dll or it will not work.

this zip containes Source for a sample project, the Flash movie source, the compiled flash movie, the compiled dll, and the dll source code.
